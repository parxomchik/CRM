 При переходе на сайт crm пользователь попадает на промо страницу

 На этой странице он может:
  - ознакомится с условиями програмы
  - зарегистрироватся
  - логин
  - востановить пароль

  -------------------------------------------------------------------------------------------------------------------------------------------

 При открытии формы логина пользователь переходит по ссылке на форму востановления пароля:

  -------------------------------------------------------------------------------------------------------------------------------------------

 Отправка формы востановление пароля

 url: api/resetpassword
 method: post

 sent_data:

 {
   smsPassword:number
   mobilePhone: number
   password: text
 }


 Если  номер телефона не зарегистрирован - выдать ответ с кодом 400 и сообщением ошибки


 Если при отправке smsPassword = undefinded, то выслать пароль на телефон пользователя
 и отправить ответ с текстом сообщения для пользователя со статус кодом 200


 Если пришли все 3 поля (smsPassword:number mobilePhone: number password: text ) - изменяем пароль и сразу логиним пользователя.

 !!! Ограничение - отправка только одного смс в день на одного пользователя !!!

   -------------------------------------------------------------------------------------------------------------------------------------------

    При успешной операции востановления пароля пользователю присваивается токен ид и на стороне клиента осуществляется переход на страницу профайла пользователя

    и отправляется запрос с /api/profile (method: get)

    после чего в received_data получаю заполненую информацию cardNumber и MobilePhoneNumber пользователя.

    -------------------------------------------------------------------------------------------------------------------------------------------


    Обязательные поля для заполнения:

            FirstName
            LastName
            SurName
            BirthDay

    После заполнения анкеты пользователь отправляет запрос /api/profile (method: post)

    Если не все обязательные поля заполнены ответить статус кодом 400 и сообщением ошибки.
