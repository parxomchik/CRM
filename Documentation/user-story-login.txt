 -------------------------------------------------------------------------------------------------------------------------------------------

 При переходе на сайт crm пользователь попадает на промо страницу

 На этой странице он может:
  - ознакомится с условиями програмы
  - зарегистрироватся
  - логин
  - востановить пароль

 -------------------------------------------------------------------------------------------------------------------------------------------

 При открытии формы логина пользователь воодит поля

  mobilePhone: number
  password: text
  privatePolicy: boolean

  После чего идет отправка формы логина:

  -------------------------------------------------------------------------------------------------------------------------------------------

 Отправка формы логина

 url: api/login
 method: post

 sent_data:

 mobilePhone: number
 password: text
 privatePolicy: boolean


 Если мобильный телефон и пароль подходят - выдать ответ с кодом 200 c сообщением о удачном логине
 Если  номер телефона не зарегистрирован - выдать ответ с кодом 400 и сообщением ошибки
 Если  номер телефона или пароль не совпадает  - выдать ответ с кодом 400 и сообщением ошибки


 После успешной проверки логина и пароля -  выдать токен ид данного пользователя.
 При переходе на страницы личного кабинета выдавать информацию пользователя по данному токен ид.

   -------------------------------------------------------------------------------------------------------------------------------------------

   После чего пользователь на стороне клиент переходит на страницу профайла и отправляется запрос с /api/profile (method: get)

    -------------------------------------------------------------------------------------------------------------------------------------------


    Обязательные поля для заполнения:

            FirstName
            LastName
            SurName
            BirthDay

    После заполнения анкеты пользователь отправляет запрос /api/profile (method: post)

    Если не все обязательные поля заполнены ответить статус кодом 400 и сообщением ошибки.


    -------------------------------------------------------------------------------------------------------------------------------------------
