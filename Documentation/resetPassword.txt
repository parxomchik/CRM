Отправка формы востановление пароля

url: api/resetpassword
method: post

sent_data:

{
  smsPassword:number
  mobilePhone: number
  password: text
}


Если  номер телефона не зарегистрирован - выдать ответ с кодом 400 и сообщением ошибки


Если при отправке smsPassword = undefinded, то выслать пароль на телефон пользователя
и отправить ответ с текстом сообщения для пользователя со статус кодом 200


Если пришли все 3 поля (smsPassword:number mobilePhone: number password: text ) - изменяем пароль и сразу логиним пользователя.

!!! Ограничение - отправка только одного смс в день на одного пользователя !!!
